The site's history

This site was created on 9/6/2023 as a project for my web programming class. While making it, I started learning a bit of Javascript. While JS isn't especially difficult, using it in conjunction with html/css is. Despite all of this, I still enjoyed putting this site together.

Purpose of the site

While I had created this site for a class project, the reason I chose to create a Flash site specifically is because I was amazed at how far Ruffle came since the Flash end of life. Not only has Ruffle been able to run many of my childhood favorite Flash games, It also has been able to make them look better. My hope was to be able to show off these games running on a modern Flash implementation and maybe even help people enjoy these games once more.

How the site works

This site is able to run Flash games due to an open source project called Ruffle which is a complete reimplementation of Adobe's Shockwave Flash Player. To better understand the difference between Flash and Ruffle, I have created a table.

Flash vs. Ruffle
Flash Ruffle
Time period 1996-2020* 2019-present
Language ActionScript Rust
Security Poor Good
*While public support for Flash ended in 2020, companies that had enterprise contracts with Adobe recieved support beyond 2020.

Due to the switch to Ruffle, a site like this can run on many more platforms than Flash player could. These include but are not limited to:

  • Android
  • IOS
  • Windows
  • MacOS
  • Linux
